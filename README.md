## Installation

1. Clonar el proyecto
2. Copiar el env.example y renombrar a .env
3. Instalar las dependencias

```bash
yarn install
```
4. Levantar la imagen (Docker Desktop)

```bash
docker-compose up -d 
```
5. Levantar el backend de Nest
```bash
yarn start:dev
```
6. Visitar el sitio
```bash
localhost:3000/graphql
```
